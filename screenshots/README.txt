# Screenshots Directory

Place your application screenshots in this directory.

## Recommended Screenshots

1. **Home Page**: Screenshot of the main application interface
2. **Team Selection**: Screenshot of the team selection interface
3. **Results Page**: Screenshot showing a generated Dream 11 team
4. **Statistics View**: Screenshot of player statistics

## Instructions

1. Take screenshots of your application in action
2. Name them descriptively (e.g., `home_page.png`, `team_selection.png`)
3. Update the main README.md to point to your primary screenshot
4. Ensure screenshots are clear and showcase the application's features

## Image Guidelines

- Use PNG format for better quality
- Keep file sizes reasonable (compress if needed)
- Recommended resolution: 1280x720 or higher
- Ensure no sensitive information is visible in screenshots
